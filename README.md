# Custom Unreal Engine Class Templates

Welcome to the **Custom Unreal Engine Class Templates** repository! This project provides cleaner and more organized default C++ class templates for Unreal Engine. 

By default, Unreal Engine generates classes with boilerplate comments that many developers end up deleting anyway. These templates remove that unnecessary clutter and provide a custom, streamlined class layout to speed up your workflow. You are completely free to modify these templates further to suit your own coding standards.

### ⚠️ Important Warnings

> * **Backup Your Files:** Before proceeding, it is highly recommended that you back up the original `.template` files located in the target directory. 
> * **Engine Verification:** If you ever use the "Verify" option for Unreal Engine in the Epic Games Launcher, or if you update the engine, these templates will be overwritten by the defaults. You will need to repeat the installation process.

### Installation Guide

**Step 1:** Ensure that Unreal Engine is completely closed.
**Step 2:** Download or clone this repository to your computer.
**Step 3:** Copy the downloaded `.template` files.
**Step 4:** Navigate to your Unreal Engine installation directory and paste the files into the following path (replace the files when prompted):

`C:\Program Files\Epic Games\UE_5.6\Engine\Content\Editor\Templates`

*(Note: If your engine is installed on a different drive or you are using a different version, adjust the path accordingly.)*
